CSS has Three Diffrent Types
  1. Inline CSS:- Applied directly in an HTML element using (style) attribute.
  2. Internal CSS:- Placed inside a < style> tag in the < head> of a single HTML page.
  3. External CSS:-Written in a separate .css file and linked via in HTML
